# Letter-sound recordings

Drop MP3 files in this folder and the Phonics tab plays them automatically.
Any sound without a file here falls back to the computer voice.

## File names

For each sound use its id (the letters shown on the tile):

- `<id>.mp3` — the pure sound on its own (plays for "🔊 Just the sound")
- `<id>-chant.mp3` — the song-style chant, e.g. "s, s, snake!" (plays when a
  letter is tapped and for the 🎵 button)

Examples: `s.mp3`, `s-chant.mp3`, `sh.mp3`, `ai-chant.mp3`

## All sound ids

Group 1: s, a, t, i, p, n
Group 2: c, k, e, h, r, m, d
Group 3: g, o, u, l, f, b
Group 4: ai, j, oa, ie, ee, or
Group 5: z, w, ng, v, oo-long, oo-short
Group 6: y, x, ch, sh, th-unvoiced, th-voiced
Group 7: qu, ou, oi, ue, er, ar
Group 8: ay, ea, igh, ow, oy, ir
Group 9: ur, aw, ew, air, ear, ph

Tips: keep clips short (under ~3 seconds), trim silence at the start so they
feel snappy, and reload the page after adding files.
